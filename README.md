## 📋 Project Overview
This project focuses on understanding **why customers are leaving** and providing actionable insights to improve retention. It analyzes ~7,000 customers from a Telco dataset.

### 🎯 Key Features
- **KPIs**: Overall Churn Rate, Total Customers, Average Tenure, Average Monthly Charges
- **Churn Analysis**:
  - By Contract Type (Month-to-Month vs 1/2 Year)
  - By Customer Tenure (new vs long-term customers)
  - By Payment Method & Internet Service
- **Interactive Filters**: Contract, Internet Service, Payment Method, Senior Citizen, etc.

## 🛠️ Tools & Technologies
- **Tableau Desktop / Tableau Public** — Dashboard development & visualizations
- **Excel + Power Query** — Data cleaning and preparation
- Business Metrics & Retention Analysis

## 📊 Key Insights
- Month-to-Month contracts have dramatically higher churn (**42.7%**) compared to longer contracts.
- New customers (0-6 months) churn at the highest rate (**54.3%**).
- Fiber Optic users churn at **41.9%** — significantly higher than DSL.
- Electronic Check payment method shows the highest churn rate (**45.3%**).

**Business Recommendation**: Prioritize retention efforts on new customers with Month-to-Month contracts and Fiber Optic service.
